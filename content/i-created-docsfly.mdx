## What is Docsfly?

Docsfly is a documentation framework built with Next.js and shadcn/ui. Using templates, you can create beautiful documentation in an instant.

### Technology Stack

- Next.js
- Tailwind CSS
- shadcn/ui
- TypeScript
- Lucide

## How to Use Docsfly

It's very simple. Just run the following command and answer a few questions to create it.

```bash
npx create-docsfly-app # replace it on your package manager
```

Then, write your documentation in the docs folder. That's it!

## Why Docsfly was developed

I wanted a modern, feature-rich docs framework. I wanted a modern docs framework built with React, Next.js, and shadcn/ui.
