## What is tracking-tight?

Tracking-tight is a CSS property that controls the tracking (letter spacing) of text. It is a value that can be set to a positive or negative number.

## Why is tracking-tight important?

Tracking-tight is important because it can help to improve the readability of text. It can also help to improve the design of a website.

## How to use tracking-tight?

It is easy to use. Just add the `tracking-tight` class to the text you want to style. (Tailwind CSS)

```tsx
<p className="tracking-tight">Hello, World!</p>
```

### For pure CSS

```css
.tracking-tight {
  letter-spacing: -0.025em;
}
```
